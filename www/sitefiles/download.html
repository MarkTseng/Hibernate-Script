TITLE=Software Suspend - Obtaining It

###TITLE###

Development versions

The latest development versions are posted to the mailing lists and also potentially are in the Berlios Download Page, near the bottom under "Test Patches". If you use these patches, you should keep up to date with the mailing list for any known problems that may arise.

Stable versions

Software suspend comes as pair of patches - one version specific patch which is particular to a particular kernel version. This must be applied first. The second is the core patch which is the same for all kernel versions, and must be applied second. The HOWTO has detailed instructions on patching and recompiling your kernel.
