TITLE=Software Suspend - Optional Extras

###TITLE###

"Options" patches extend the functionality of Software Suspend, but depend on other patches which are not normally found in the vanilla kernels.

For the 2.1 release of Software Suspend, there are no available option patches, largely because most of the code they depended on has been merged into the mainstream kernel and support has been merged into the main release of Software Suspend (eg, XFS, ACPI, Laptop mode). Bootsplash option patches may become available soon.

The following information was current for version 2.0 of swsusp.

Bootsplash

Bootsplash support allows you to have a graphical boot process for your Linux machine. The optional patch for software suspend (called bootsplash-option) will make software suspend utilise the progress bar of bootsplash rather than the default text progress screen that is normally shown. It is only required if you are using version 1 or 2 bootsplash images. If you have a version 3 bootsplash image, you do not need this patch. Apply this option patch after applying the one from bootsplash.org

Win4Lin

Win4Lin is a commercial program that allows you to run Windows(TM) under Linux. The patch-swsusp-win4lin patch allows you suspend whilst running Win4Lin.
