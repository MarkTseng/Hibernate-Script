TITLE=Software Suspend - Optional Extras

###TITLE###

There are several "Options" available for software suspend - these patches extend the functionality, but depend on other patches which are not normally found in the vanilla kernels. These patches are contained within the swsusp tarball obtained from the download page.

This information is current for version 2.0 of swsusp.

ACPI option is no longer needed.

Bootsplash

Bootsplash support allows you to have a graphical boot process for your Linux machine. The optional patch for software suspend (called bootsplash-option) will make software suspend utilise the progress bar of bootsplash rather than the default text progress screen that is normally shown. It is only required if you are using version 1 or 2 bootsplash images. If you have a version 3 bootsplash image, you do not need this patch. Apply this option patch after applying the one from bootsplash.org

Win4Lin

Win4Lin is a commercial program that allows you to run Windows(TM) under Linux. The patch-swsusp-win4lin patch allows you suspend whilst running Win4Lin.

Laptop mode (description) (latest original patch)

NOTE: Laptop mode has been integrated in the 2.4 kernel series since 2.4.23 and this is no longer required if you are using 2.4.23 or later.

Laptop mode is a patch written by Jens Axboe that adds a noticeable amount of battery life to a laptop, by holding data to be written to the disk until a certain amount of time has elapsed or a read is required - thus not spinning up the hard disk unnecessarily. The laptop-option patch is the modified version of the patch that integrates with software suspend. This option patch does not require the original patch be applied to the kernel.
WARNING: Using this patch increases the risk of data loss in the case of unexpected power, software or hardware failures.

XFS support

If you have XFS partitions, then you will need this patch to be able to suspend properly. The patches are created against recent snapshot versions of the XFS code by Karol Kozimor, and must be applied after the XFS patch, but before software suspend. Download it from here
